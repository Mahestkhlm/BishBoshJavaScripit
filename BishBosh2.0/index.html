Bish-Bosh 2.0

SOLUTION & METHOD:

A very small application to run BISH-BOSH GAME! Prints out numbers 1 to 100, Replace all numbers divisible by 3 with “Bish”, Replace all numbers divisible by 4 with “Bosh”, replace all numbers divisible by both 3 and 4 with “bishbosh”

step1: created a function myfunction() with extended arguments.Iterartive loops step 3:The first % division will divide the current loop value, and if it is evenly divided it will display "Bish" step 4:The second division is just as the same like previous step, it will display"Bosh" If both first and second % value is evenly divided then it will display "Bish-Bosh". The number will be displayed if none of the conditions are true
